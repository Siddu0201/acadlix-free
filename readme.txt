== Description ==

Acadlix is a powerful Learning Management System (LMS) plugin for WordPress designed to create and manage online courses with ease. Whether you are an educational institution, a business looking to train employees, or an individual offering courses, Acadlix provides all the tools you need to deliver engaging and interactive learning experiences.

= Features =
* Create and manage unlimited courses
* Organize courses into categories and tags
* Create lessons and quizzes
* Track student progress
* Issue certificates of completion
* Integrate with popular payment gateways
* User-friendly interface
* Fully responsive design

== Installation ==

1. Upload the `acadlix` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Follow the setup wizard to configure the plugin.

== Frequently Asked Questions ==

= How do I create a new course? =
To create a new course, navigate to the Acadlix menu in your WordPress dashboard and click on 'Add New Course'. Fill in the necessary details and publish your course.

= Can I issue certificates to students? =
Yes, Acadlix allows you to create and issue certificates to students upon completion of courses.

= Is Acadlix compatible with my theme? =
Acadlix is designed to work with most WordPress themes. However, for the best experience, we recommend using a theme optimized for LMS.
